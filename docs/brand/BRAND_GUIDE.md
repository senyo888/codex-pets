# Codex Pets Brand Guide

Codex Pets is the public name of this independent community catalogue. Use it
proudly, clearly, and consistently.

## Identity

The brand mark is the **orbital paw**: four crystalline toe facets and one
faceted paw pad held inside an open orbit, finished by a small gold square node.
It combines companionship, motion, and inspectable digital craft without
borrowing a platform vendor's identity.

The name is always written as **Codex Pets**. Do not shorten it in public titles
or merge it into a single word.

The logo is an RGBA PNG with a transparent background. The header intentionally
contains no rasterized words; surrounding HTML or Markdown owns the accessible
**Codex Pets** title.

## Presentation density

- Keep the website navigation bar compact: a `58 px` desktop minimum height with a `36 px` mark.
- Use the wide header as atmosphere, not as a second title. The website hero supplies the real heading and supporting copy.
- In the repository README, display the wide header at `92%` width and do not stack a second standalone logo beneath it.
- Preserve the source asset ratio. Any future crop or replacement requires its own visual review across the website hero and GitHub README.

## Colour

| Role | Value |
| --- | --- |
| Deep navy | `#06111f` |
| Luminous cyan | `#64e6f5` |
| Electric blue | `#1787ff` |
| Restrained violet | `#8b5cf6` |
| Warm gold accent | `#f9c642` |
| Primary text | `#f0fbff` |
| Muted text | `#9cb2c3` |

Deep navy is the foundation. Cyan and blue carry the identity; violet adds
depth; gold is a small signal accent, never the dominant colour.

## Typography

Use a modern system sans-serif stack. Titles should be bold, tightly tracked,
and high contrast. Keep **Codex Pets** as real text instead of baking it into an
image, so the name remains selectable, accessible, searchable, and resilient at
every viewport size.

## Spacing and minimum size

- Keep clear space around the logo equal to at least one toe-facet width.
- Use the mark at `32 px` or larger in digital interfaces.
- Use the complete mark; do not crop the orbit or detach the gold node.
- On busy imagery, place the mark on deep navy or use the supplied header rather
  than adding an improvised glow or outline.

## Do not

- redraw, stretch, rotate, recolour, or add text inside the logo;
- use the mark as a replacement for a pet's authoritative package artwork;
- add vendor logos, knot-like vendor geometry, or endorsement language;
- place the mark on low-contrast backgrounds;
- regenerate pet spritesheets or package assets to match the brand artwork.

## Independence and truth

Keep the independent-community disclaimer visible wherever platform context
could be misunderstood. Brand presentation may change; package metadata,
checksums, installer targets, and validation evidence remain authoritative.
